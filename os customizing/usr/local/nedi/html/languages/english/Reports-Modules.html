Module Reports

Need to know how many modules of a kind you've got? This report also helps, if you need to generate a HW inventory for support contracts based on serial numbers etc.

DistributionPresents an overview of which modules are installed in which devices
InventoryGenerates a complete list of devices and their individual modules
Print suppliesLists print supplies sorted by availability or location (to make filling them up easier for the guy who has to go to every printer)
Virtual MachinesList all hypervisors with allocated VMs, CPUs and memory
